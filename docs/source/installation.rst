============
Installation
============


Stable release
--------------

To install keywords_extractor, run this command in your terminal:

.. code-block:: console

    $ pip install keywords_extractor

This is the preferred method to install keywords_extractor, as it will always
install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for keywords_extractor can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/Krasherr/keywords_extractor

Or download the `tarball`_:

.. code-block:: console

    $ curl  -OL https://github.com/Krasherr/keywords_extractor/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/Krasherr/keywords_extractor
.. _tarball: https://github.com/Krasherr/keywords_extractor/tarball/master
